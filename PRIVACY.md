# Privacy Policy for Redline

Last updated: July 6, 2026

Redline ("the Application") is a lightweight desktop image annotation tool. This Privacy Policy describes how we handle information when you use the Application.

## 1. Personal Information Collection
We do not collect, store, transmit, or share any personal information or usage data. 

*   **No Tracking:** The Application does not contain any analytics, telemetry, tracking SDKs, or advertisements.
*   **No Accounts:** You do not need to create an account or sign in to use the Application.

## 2. Image and File Data
All image annotation, cropping, resizing, and file operations are performed entirely locally on your device. 

*   **No Cloud Uploads:** Your images, screenshots, and annotations never leave your computer. We do not upload your files to any cloud server or third-party service.

## 3. Network Connections
The Application operates offline by default. If you use features that check for updates, the Application may make a request to check for new releases:
*   On Windows (via the Microsoft Store), updates are managed entirely by the Microsoft Store client.
*   The Application may query GitHub to check for the latest release versions. No personal identifying information is transmitted during this check.

## 4. Contact
If you have any questions or concerns about this Privacy Policy, please open an issue on our [GitHub Issues](https://github.com/jarrodek/redline/issues) page.
